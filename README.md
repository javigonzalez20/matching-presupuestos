# Matching Presupuestos

Repositorio del Trabajo Fin de Máster centrado en el emparejamiento semántico de conceptos técnicos en presupuestos históricos de construcción.

El objetivo del proyecto es recuperar información relevante de materiales usados en proyectos anteriores, incluso cuando los conceptos están redactados de forma distinta, y generar una hoja de Excel con una primera estimación para nuevos proyectos.

## Contenido
- Evaluación de métodos léxicos y semánticos (TF-IDF, embeddings y cross-encoders)
- Implementación de un sistema híbrido con regla de medida
- Prototipo final para lectura de presupuestos en Excel y generación automática de salida

## Mejor configuración obtenida
- Embedding: `multilingual-e5-base`
- Cross-encoder: `ms-marco-MiniLM-L-12-v2`
- Selección final: coincidencia de medida
